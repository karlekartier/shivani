# A. Shivani - Professional Portfolio

Professional portfolio website for **A. Shivani**, a Medical Writer and clinical research specialist.

## Features

- **Modern & Responsive Design**: built with a focus on medical accuracy and professional aesthetics.
- **Dynamic Theme Switching**: Seamless toggle between Light and Dark modes.
- **Optimized for Reading**: Typography and contrast optimized for scientific and medical content.
- **Research Showcase**: Integrated project filter system to categorize clinical research and oncology studies.
- **Interactive Experience**: Smooth animations and transitions powered by custom semantic CSS.

## Tech Stack

- **HTML5 & Vanilla CSS**: For structure and layout.
- **Tailwind CSS**: For rapid utility-based styling.
- **Bootstrap Icons & Boxicons**: For unified, professional iconography.
- **Storyset Illustrations**: Customized medical-themed illustrations.

## Project Structure

```text
├── assets/             # Images, PDFs, and Stylesheets
│   ├── css/            # Semantic style.css and Tailwind output
│   ├── img/            # Illustrations and project images
│   └── pdf/            # Resume and credentials
├── index.html          # Main landing page
└── README.md           # Project documentation
```

## Deployment

To view the project locally, simply open `index.html` in any modern web browser.
